# nea-data-sg

NEA data Singapore

maven project, console application in java

## Requirement:

 - install java (1.8+)

## Testing and run:

```
// build jar
$ ./mvnw package

// run jar
$ java -jar /target/nea-data-sg-1.0.0.jar


// start project - run Main.java

// run test case - run MainTest.java
```

open project in intellij idea
